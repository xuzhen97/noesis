# Noesis 灵识

Noesis 灵识 is a personal Human-AI Symbiotic Workspace. 本仓库是新实现的独立项目根目录。

## 当前范围

本 workspace 刻意保持小规模。初始化阶段只创建 P0 控制闭环骨架，围绕以下概念：

- Gateway
- Client Agent
- Machine
- Task
- Task Event

## 开发

```bash
pnpm install
pnpm build
pnpm test
```

## 项目语言

业务文档默认使用简体中文。代码标识符、包名、协议字段、数据库字段、枚举值使用英文。
