Noesis Distribution 架构

P0 控制闭环 · Gateway → Client Agent → CLI/SDK

HTTP API 路由

  • • GET /api/health — 健康检查
  • • POST /api/tasks — 创建任务
  • • GET /api/tasks/:id — 查询任务
  • • GET /api/tasks/:id/events — 事件列表

WebSocket 消息

  • • client.hello — Agent 注册
  • • client.accepted — 注册确认
  • • task.dispatch — 任务派发
  • • task.event — 事件上报

Client Agent 能力

  • • WS 自动连接 + 注册
  • • 白名单命令执行 (spawn)
  • • stdout/stderr 捕获上报
  • • 等待队列自动派发

安全约束

  • • 白名单仅允许 node -e console.log
  • • 错误消息不泄露 stack/路径
  • • Task Event 只追加不删除
  • • Gateway 无持久化（内存状态）
